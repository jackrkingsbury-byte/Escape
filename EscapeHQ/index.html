Automation Flows
💰 New Sale → Make.com
When you log a sale, fire a Make.com webhook to trigger your post-purchase automation (send code, tag CRM, send Slack notification, etc.)
🤖 Agent Output → n8n
After each agent run completes, forward the output to your n8n workflow for further processing, storage, or publishing.
📣 Daily Marketing → Relevance AI
Route the Marketing Agent's output to a Relevance AI agent for tone refinement, social scheduling, or additional AI steps.
Quick Setup Guides
⚡ Make.com
1. Create a new Scenario
2. Add a Webhooks → Custom webhook trigger
3. Copy the webhook URL here
4. Add your automation steps (Gmail, Notion, Slack…)
🔀 n8n
1. Create a new workflow
2. Add a Webhook trigger node
3. Copy the production webhook URL
4. Build your workflow (Airtable, Discord, etc.)
🧠 Relevance AI
1. Go to relevanceai.com and create an Agent
2. Copy your API Key from Settings
3. Copy the Agent ID from the agent URL
4. Select your region and click Trigger
