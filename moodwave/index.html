FAQ

Good questions

Do I need a Spotify account?

No — the Studio works in demo mode with zero setup, so you can plan playlists straight away. You only connect Spotify when you want to save a playlist to your real account and press play.

How does "mood" actually work?

Each mood is mapped to a target for energy, positivity (valence) and tempo, plus a set of genre and keyword cues. MoodWave scores tracks against that target and against the artists you picked, then orders them so the playlist flows — building or winding down to match the feeling.

Can I use any artist, like Travis Scott?

Yes. Tap a preset or type any artist name. When you're connected to Spotify we pull that artist's real catalogue live; in demo mode we use a built-in library of popular artists so you can see how it feels.

Is my Spotify login safe?

Completely. We use Spotify's official PKCE login — the same standard big apps use. You log in on Spotify's own page, we never see your password, and nothing is ever posted to your account unless you tap Save.

I want to sell this. Can I?

That's the idea. It's a single self-contained file you own outright. Rename it, restyle it, plug in your own payment link on the pricing buttons, add your own Spotify app credentials, and ship it. The README walks through going live.
